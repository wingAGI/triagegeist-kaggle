# Data Access

This repository does not redistribute the Triagegeist competition data.

To run the full pipeline:

1. Join the competition on Kaggle.
2. Download the competition files.
3. Place the files either:
   - in `data/raw/`
   - or as `data/triagegeist.zip`

Expected files:

- `train.csv`
- `test.csv`
- `chief_complaints.csv`
- `patient_history.csv`
- `sample_submission.csv`

The Kaggle-ready notebook can also read directly from `/kaggle/input/triagegeist` inside Kaggle.
